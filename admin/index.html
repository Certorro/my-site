Социальные сети

Экспорт / Импорт данных

Сохраните все данные в JSON-файл для резервной копии или импортируйте ранее сохранённые данные.

Статические HTML-страницы

Скачать HTML-файлы всех статей и профилей адвокатов для размещения на сервере.

После скачивания: поместите файлы статей в папку articles/, профили адвокатов в папку team/, затем выполните git push.
